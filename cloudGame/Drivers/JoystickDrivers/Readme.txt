Normally this driver is installed by the installer.  If you need to do a manual install or uninstall, batch files are included to do this.

Note 1: Run the .bat files as Admin!

Note 2: Make sure that any apps using these drivers are not running while you install or uninstall the drivers!

INSTALLING
To install the 'Tetherscript Virtual Joystick' driver
1) Run the install.bat as admin.  This batch file will attempt to first remove the driver, then install it.  If it is successful, you will see the driver listed in the Windows Device Manager.

UNINSTALLING
To uninstall the 'Tetherscript Virtual Joystick driver
1) Run the uninstall.bat as admin.  If it is successful, you will no longer see the driver listed in the Windows Device Manager.
